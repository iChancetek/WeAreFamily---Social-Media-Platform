# Famio Brand Assets

This folder contains official Famio branding assets for download and distribution.

## Available Assets

### App Icons (PNG)
- `famio-icon-64.png` - 64×64px (Small icons, favicons)
- `famio-icon-128.png` - 128×128px (Standard web use)
- `famio-icon-192.png` - 192×192px (PWA launcher icon)
- `famio-icon-512.png` - 512×512px (High-resolution, splash screens)

## Usage Guidelines

### Brand Colors
- **Primary Blue**: `#0057B7` (LinkedIn-inspired professional blue)
- **Background**: White or light gray
- **Text**: Dark gray or black

### Logo Usage
- Maintain clear space around the logo (minimum 20% of logo width)
- Never distort or skew the logo
- Use official color versions when possible
- For dark backgrounds, use white/light version

### Contexts
- **Navigation**: Use 64px or 128px versions
- **Hero/Landing**: Use 192px or 512px versions  
- **Social Media**: Use 512px version
- **Favicons**: Use 64px version

## Download Instructions

Right-click any PNG file → "Save As" to download to your desktop.
